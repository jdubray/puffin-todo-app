# Pulse — Documentation Index

> *Tasks that live, breathe, and remember.*

## What is Pulse?

Pulse is a desktop PWA to-do application built around a radical idea: **tasks are not checkboxes, they are living things.** They grow with context, they remember where they've been, they connect to the world you work in, and they disappear gracefully when their purpose is fulfilled.

This is not another productivity app. It is a personal command center for your attention.

## Quick Reference: Guiding Principles

1. **Personal** — It knows you. It learns your patterns. It feels like yours.
2. **Innovative** — Tasks are entities, not items. The interface is spatial, not linear.
3. **Productive** — Ruthless about reducing friction. Everything is one gesture away.
4. **Fun** — Completing work should feel rewarding. The app should spark joy.
